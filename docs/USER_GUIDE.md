## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- JavaScript enabled
- Mobile app available on iOS and Android

### First Time Login
1. You'll receive login credentials from admin
2. Visit: https://app.eliteifitness.com
3. Enter your email and password
4. Set up two-factor authentication (recommended)
5. Update your profile information

## Member Management

### Adding a New Member

1. Go to **Members** → **New Member**
2. Fill in member details:
   - Full name, email, phone
   - Date of birth, gender
   - Address information
   - Emergency contact
   - Blood group (optional)
3. Select membership package
4. Assign trainer (optional)
5. Set membership dates
6. Click **Create Member**
7. System generates Member ID and QR code

### Editing Member Information

1. Go to **Members**
2. Search for member by name or ID
3. Click on member name
4. Edit required fields
5. Click **Save Changes**

### Member Statuses
- **ACTIVE**: Current, active membership
- **INACTIVE**: Paused membership (on hold)
- **SUSPENDED**: Membership frozen (payment issues)
- **EXPIRED**: Membership ended

### Bulk Member Import

1. Go to **Members** → **Import**
2. Download template Excel file
3. Fill in member data
4. Upload file
5. Review and confirm
6. System imports members

## Payments & Billing

### Processing a Payment

1. Go to **Payments** → **New Payment**
2. Select member
3. Enter payment amount
4. Select payment method:
   - Card (Stripe)
   - Cash
   - Check
   - UPI
   - Net Banking
5. Add optional notes
6. Click **Process Payment**
7. For card payments, confirm on Stripe
8. Receipt is automatically sent

### Payment Methods
- **Card**: Immediate processing via Stripe
- **Cash**: Manually mark as completed
- **Check**: Verify and mark as cleared
- **UPI/Net Banking**: Verify transaction ID

### Refunds

1. Go to **Payments**
2. Find transaction
3. Click **Refund**
4. Enter refund reason
5. Confirm refund
6. Refund processed within 3-5 business days

### Membership Renewal

1. Go to **Members** → find member
2. Click **Renew Membership**
3. Select new package
4. Process payment
5. New membership dates are updated

### Payment Reminders

System automatically sends:
- Email and SMS reminders 7 days before due date
- Membership expiry notices 30 days before expiry
- Payment confirmation receipts

## Trainers & Scheduling

### Adding a Trainer

1. Go to **Trainers** → **New Trainer**
2. Enter trainer details:
   - Name, email, phone
   - Specialization (e.g., Yoga, CrossFit, Pilates)
   - Years of experience
   - Certifications
   - Join date
   - Shift preference
3. Click **Add Trainer**

### Assigning Trainer to Member

1. Go to **Members**
2. Find and click member
3. Click **Assign Trainer**
4. Select trainer from list
5. Click **Confirm**
6. Member can now book sessions

### Trainer Schedule

1. Go to **Trainers** → select trainer
2. View weekly schedule
3. Add sessions:
   - Select member
   - Pick day and time
   - Add notes (optional)
4. Sessions appear on both trainer and member calendars

## Attendance Tracking

### Member Check-In

1. At reception: **Check-In** screen
2. Option 1: Scan QR code with mobile
3. Option 2: Enter member ID manually
4. Member name and photo appears
5. Click **Check-In**
6. Attendance recorded automatically

### QR Code Scanning

- Requires mobile app or QR scanner
- QR codes printed on membership cards
- Can also use digital QR in member app

### Attendance Analytics

1. Go to **Reports** → **Attendance**
2. View by:
   - Date range
   - Individual member
   - Overall gym usage
   - Peak hours
3. Export reports as CSV/PDF

### Member Streaks

- Automatically tracked
- Displayed on member profile
- Used for gamification/rewards

## Troubleshooting

### Can't Log In

1. Check internet connection
2. Verify email address
3. Reset password: Forgot Password link
4. Check if account is suspended
5. Contact admin if still issues

### QR Code Not Scanning

1. Ensure QR code is not damaged
2. Clean camera lens
3. Ensure adequate lighting
4. Try manual ID entry instead
5. Request new QR code from admin

### Payment Failed

1. Check internet connection
2. Verify card details
3. Ensure sufficient funds
4. Try alternative payment method
5. Contact bank or use UPI/Net Banking

### Member Not Found

1. Check spelling of name
2. Try searching by phone
3. Try searching by member ID
4. Check if member status is suspended
5. Search in archive if old member

### Missing Attendance Records

1. Check date range filter
2. Verify member is checked in
3. Look in "Pending" tab
4. Check if system was down
5. Contact admin with screenshot
